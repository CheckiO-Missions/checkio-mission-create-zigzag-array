This function creates an {% if interpreter.slug == "js-node" %}Array of arrays.{% else %}List of lists.{% endif %} that represents a two-dimensional grid with the given number of rows and cols. This grid should contain the integers from start to start + rows * cols - 1 in ascending order, but the elements of every odd-numbered row have to be listed in descending order, so that when read in ascending order, the numbers zigzag through the two-dimensional grid.

Input: Two ints rows and cols. One optional argument start.

Output: {% if interpreter.slug == "js-node" %}Array of arrays.{% else %}List of lists.{% endif %}

Example:

{% if interpreter.slug == "js-node" %} 
createZigzag(3, 5) == [
        [1,2,3,4,5],
        [10,9,8,7,6],
        [11,12,13,14,15]
    ]
createZigzag(5, 1) == [
        [1],
        [2],
        [3],
        [4],
        [5]
    ]
createZigzag(2, 3, 5) == [
        [5, 6, 7],
        [10, 9, 8],
        [11, 12, 13]
    ]
{% else %}
